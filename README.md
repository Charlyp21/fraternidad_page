# fraternidad_page

Sitio web multipagina para el club universitario La Fraternidad.

## Estructura
- index.html
- paginas/acerca.html
- paginas/comida.html
- paginas/lugares.html
- paginas/unete.html
- assets/styles.css
- assets/main.js

## Uso
Abre index.html en tu navegador y navega entre secciones usando el menu superior.
