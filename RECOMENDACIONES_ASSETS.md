# Recomendaciones de assets para estilo medieval

## Assets sugeridos
- Textura de pergamino en JPG o PNG para fondos de secciones.
- Iconos SVG para menu, mapa, comida y comunidad.
- Ilustracion ornamental (borde o sello) para hero o footer.
- Version cuadrada del logo para avatar/redes.

## Estructura recomendada
Guarda nuevos recursos en:
- assets/images/
- assets/icons/

## Integracion rapida
1. Agrega imagenes en `assets/images`.
2. Actualiza rutas en HTML/CSS con formato relativo, por ejemplo:
   - HTML: `./assets/images/pergamino.jpg`
   - CSS: `url("./images/pergamino.jpg")` cuando el CSS esta en `assets/`
3. Optimiza archivos para web:
   - JPG/WebP para fotos e ilustraciones grandes.
   - SVG para iconos y elementos lineales.

## Sugerencia de contenido futuro
- Comida: fotos de platos, logos de locales, iconos de precio.
- Lugares: mini mapas, fotos de edificios, marcadores de ruta.
- Unete: insignias o escudos para reforzar identidad visual.
